Prevents wordpress from being redirected to canonical url. this plugin makes any accessible doamin to your wordpress work separately.

== Description ==

If you have a Wordpress blog with multiple addresses like this:

* http://example.com/
* http://www.example.com/
* https://example.com/
* https://www.example.com/

Every time you visit one of alternatives, Wordpress redirects you to your main URL, even when you prevent redirection, all links are absolute of main address, this plugin makes them work separately.

== Installation ==

1. Upload the `no-redirect` folder to the `/wp-content/plugins/` directory
1. Activate the No Redirect plugin through the 'Plugins' menu in WordPress
